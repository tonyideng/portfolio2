Interactive Web Prototype

Project Scope

This project was completed as part of the course Programmable User Interfaces at CMU, I was tasked with creating a working prototype of a To-Do List Mobile App. The project started with Lo-Fi Paper Prototypes, then transitioned into Mid/Hi-Fi screen designs in Sketch and Invision. Finally it culminated in an interactive web prototype optimized for mobile view, built using HTML/CSS/JavaScript.
